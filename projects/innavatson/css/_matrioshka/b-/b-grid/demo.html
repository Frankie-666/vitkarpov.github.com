Отступы между ячейками

Отступы между ячейками .b-grid__cell задаются не самим ячейкам, а элементам внутри них. Можно свободно задавать отступы и в процентах и в пикселях.

Например, мы хотим сделать отступ между ячейками в 5%

HTML 

<div class="b-grid">
   <div class="b-grid__cell b-grid__cell_1-4">
       1-4
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       1-4
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       1-4
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       1-4
   </div>
</div>

Создаем новый блок .b-example и внутри ячейки .b-grid__cell элемент .b-example__block и задаем этому элементу отступ слева в 5%

HTML 

<div class="b-grid b-example">
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
</div>

CSS 

.b-example__block {
    margin-left: 5%;
}

Теперь между каждым элементом .b-example__block есть отступ в 5%, но у первого элемента он нам не нужен.

Для того чтобы компенсировать отступ первого элемента прописываем на блоке .b-example отступ слева margin-left: -5%;

HTML 

<div class="b-grid b-example">
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
   <div class="b-grid__cell b-grid__cell_1-4">
       <div class="b-example__block">1-4</div>
   </div>
</div>

CSS 

.b-example {
   margin-left: -5%;
}

.b-example__block {
    margin-left: 5%;
}
Все готово, отступы между ячейками .b-grid__cell равны 5% и при этом они выровнены по правому и левому краю.

Сокращенная нотация
Для блока можно использовать сокращенную нотацию, пример: 
.b-grid            => .b-g
.b-grid__cell      => .b-g__c {}
.b-grid__cell_1-2  => .b-g__c_1-2 {}
