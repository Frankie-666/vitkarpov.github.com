Разделитель .b-divider

  • Ширина .b-divider__side зависит от вложеного в него контента.
  • .b-divider__base занимает всю доступную ширину минус ширина .b-divider__side.
  • .b-divider__side и .b-divider__base не пересекаются.

Пример:

side
base

Состояния

Элемент .b-divider__side имеет два состояния:

  • Расположение слева по-умолчанию.
  • Расположение справа когда добавлен модификатор .b-divider__side_rt.

Пример:

side | side_rt
base

mix

Можно совмещать несколько элементов .b-divider__side в одном блоке.

Пример:

side
side
side | side_rt
base

Раскладка:

Отступы и размеры у элементов создаются с помощью модификаторов.

side | side_nav (width:20%)
side | side_rt | side_extra (width:250px)
base
.b-divider__side_nav {
    margin-right:18px;
    width:20%;
}

.b-divider__side_extra {
    margin-left:18px;
    width:250px;
}
Сокращенная нотация
Для блока можно использовать сокращенную нотацию, пример: 
.b-divider           => .b-d
.b-divider__side     => .b-d__s {}
.b-divider__side_rt  => .b-d__s_r {}
.b-divider__base     => .b-d__b {}
